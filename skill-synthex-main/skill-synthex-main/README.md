# Skill Synthex System V3: Precision & Robustness Re-engineered

A professional career growth platform powered by React + Node.js + Native ML.

## 🚀 Quick Start (Production Mode)

1. **Start the Backend Server**:
   ```powershell
   cd backend
   npm run dev
   ```
2. **Access the App**:
   👉 [http://localhost:5000](http://localhost:5000)

---

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite, Recharts, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express, MySQL, JWT Auth.
- **AI/ML**: Native JavaScript Similarity Engine (Node-Native).

## 📖 Features
- **Accurate Dashboard**: Real-time professional growth metrics.
- **Precision Assessment**: rigorous 30-question MCQs for competency verification.
- **Roadmap Enrichment**: Personalized learning paths with Critical Focus area highlights.
- **Glassmorphism UI**: High-end minimalist design for a premium experience.

## 👨‍💻 Development
To run in development mode with Hot Module Replacement (HMR):
1. Start backend (as above).
2. `cd frontend && npm run dev`.
3. Open `http://localhost:5173`.
